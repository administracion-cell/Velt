Los 6 planes de pago — ¿que significa cada uno?

Todos los planes son para el mismo depa. La diferencia es cuanto pagas de enganche y mensualidades durante la preventa (25 meses), y que descuento te dan a cambio. El resto (contraentrega) se paga al recibir el depa.

Guia rapida: ¿que plan te conviene segun el escenario?

Para cada nivel de apreciacion, te dice cual plan/depa te da el mejor ROI y cuanta ganancia. Asi sabes rapidamente: si creo que la plusvalia sera X%, me conviene el plan Y.

ROI = Ganancia neta / Dinero total de tu bolsa. Con hipoteca: anticipo + mensualidades + com. apertura + pagos hip. durante venta. Los costos de venta se desglosan: comision agente + ISR + notario.

Break-even: Apreciacion minima para no perder

Cada plan tiene un descuento distinto. Los costos de venta (agente + ISR + notario) se comen parte de ese descuento. Esta tabla te dice: ¿cuanta plusvalia minima necesitas para que la ganancia sea positiva? Si el mercado sube mas que ese %, ganas. Si sube menos, pierdes. Ordenado de mas seguro (arriba) a mas riesgoso (abajo).
